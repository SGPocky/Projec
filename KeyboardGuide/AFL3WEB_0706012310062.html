Designing and Building a RP2040 Stabless 40% Keyboard

Making a Handwired keyboard had always seemed daunting to me as it all looks difficult “building” and making, but it is actually quite simple but requires alot of time so i will show you the simplest way of building this keyboard shown in the first page of the gallery

Materials

  • RP2040 Zero x1
  • Copper wires (10 meters)
  • Keycaps x42
  • Switches x42
  • Acrylic Plates
  • Screws x8
  • Diodes x42

Reasoning Behind

RP2040 zero is one of the smallest for its price that still could make a keyboard and also is very strong making it not a problem for messing around and adding all of the features that existed. The copper wires will be used to wire the keyboard switches together to make a matrix grid which i will explain later on. The keycaps is self explanatory and you could choose any of it and this is also the same for the switches. As for the Acrylic plates this will be used to house the keyboard case with each other and you should ask your local laser printer person to print it for you by sending them a file that i linked or another case file if you would like a different design. The diodes will be used to differentiate between lots of the keys and allows the keyboard to detect more than 1 key at a time without issue.

Wiring Diagram

Cables are connected in a matrix grid like this:
The yellow line resembles the columns while the Blue line resembles the rows. As for the diodes it will be placed in the other side of the keyboard switch and connected with the blue wires on the black one facing up (Orange-black color)


Diode Direction

RP2040 PINS

Now you can pick from any GPxx Pin from the RP2040 PINS diagram that you want to use and connect it between each column and each row of the keyboard

How to Solder

In case you don't know how to solder, all you need to do is to have solder wires on your right hand(remember to have a decent amount of solder wire length between your hand and the soldering iron), and the soldering iron on your left hand. Then on the point where there needs to be a connection, you touch the soldering iron with the solder wire and then a connection would be made. To check whether the connection is made, you could try to wiggle the cable a bit, if it doesn't move then the connection is made.
